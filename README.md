# lyricly

simple CLI for grabbing lyrics of your favourite songs;

Usage:

```
lyricly airplanes in the night sky are like shooting stars
```

## Installation

### Cargo

```sh
cargo install lyricly
```

### Manual

```sh
git clone https://github.com/LeoDog896/lyricly
cd lyricly
cargo build --release
```
