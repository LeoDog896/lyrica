# lyrica

simple CLI for grabbing lyrics of your favourite songs;

Usage:

```
lyrica airplanes in the night sky are like shooting stars
```

## Installation

```sh
cargo install lyrica
```

```sh
git clone https://github.com/LeoDog896/lyrica
cd lyrica
cargo build --release
```
